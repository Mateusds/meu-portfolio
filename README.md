# Portfolio - Mateus Marques

Projeto de portfolio em React + TypeScript + Next.js (App Router).

## Rodar localmente

```bash
npm install
npm run dev
```

Acesse `http://localhost:3000`.

## Personalizacao rapida

- Atualize seu LinkedIn em `app/page.tsx` no `href` do botao `LinkedIn`.
- Atualize seu email em `app/page.tsx` no `href` `mailto:...`.
